# Dtw

# 函数简介

本函数用于计算两列数值型数据的DTW距离。

函数名： DTW

输入序列： 仅支持两个输入序列，类型均为 INT32 / INT64 / FLOAT / DOUBLE。

输出序列： 输出单个序列，类型为DOUBLE。序列仅包含一个时间戳为0、值为两个时间序列的DTW距离值。

提示：

  • 如果某行数据中包含空值、缺失值或NaN，该行数据将会被忽略；
  • 如果数据中所有的行都被忽略，函数将会输出0。
