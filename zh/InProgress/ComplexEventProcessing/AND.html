# 函数简介

本函数用于查询时间序列中具有并行关系的模式匹配，并输出匹配的个数。

函数名： SEQ

输入序列： 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE

参数：

  • WITHIN：匹配的时间序列的时间间隔的最大值。
  • ATLEAST：匹配的时间序列的时间间隔的最小值。

输出序列： 输出匹配的个数，类型为INT32。
