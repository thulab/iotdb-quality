# KSigma

# 函数简介

本函数利用动态K-Sigma算法进行异常检测。在一个窗口内，与平均值的差距超过k倍标准差的数据将被视作异常并输出。

函数名： KSIGMA

输入序列： 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE

参数：

  • k：在动态K-Sigma算法中，分布异常的标准差倍数阈值，默认值为3。
  • window：动态K-Sigma算法的滑动窗口大小，默认值为10000。

输出序列： 输出单个序列，类型与输入序列相同。

提示： k应大于0，否则将不做输出。
