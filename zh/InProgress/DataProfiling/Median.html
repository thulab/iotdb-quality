# Median

# 函数简介

本函数用于计算单列数值型数据的精确或近似中位数。

函数名： MEDIAN

输入序列： 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE

参数：

  • error：近似中位数的基于排名的误差百分比，取值范围[0,1)，默认值为0。如当error=0.01时，计算出的中位数的真实排名百分比在0.49~0.51之间。当error=0时，计算结果为精确中位数。

输出序列： 输出单个序列，类型为DOUBLE，序列仅包含一个时间戳为0、值为中位数的数据点。
