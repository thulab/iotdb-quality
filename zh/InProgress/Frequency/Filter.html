# 函数简介

本函数对输入序列使用有理传递函数进行滤波。有理传递函数为

函数名： FILTER

输入序列： 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE

参数：

  • numerator：有理传递函数的分子系数，使用逗号分隔。
  • denominator：有理传递函数的分母系数，使用逗号分隔。

输出序列： 输出单个序列，类型为DOUBLE，它是滤波后的序列，长度与时间戳均与输入一致。

提示： 输入序列中的NaN将被忽略。
