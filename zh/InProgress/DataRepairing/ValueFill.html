# ValueFill

# 函数简介

函数名： FILL

输入序列： 单列时序数据，类型为INT32 / INT64 / FLOAT / DOUBLE

参数：

  • method: “ffill"指使用前值填补方法；“linear"指使用线性填补方法；缺省情况下使用“ffill”。

输出序列： 即修复后的单维序列。

# 使用示例
