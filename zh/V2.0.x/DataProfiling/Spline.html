# Spline

# 函数简介

本函数提供对原始序列进行三次样条曲线拟合后的插值重采样。

函数名： SPLINE

输入序列： 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE。

参数：

  • points：重采样个数。

输出序列：输出单个序列，类型为DOUBLE。

提示：输出序列保留输入序列的首尾值，等时间间隔采样。仅当输入点个数不少于4个时才计算插值。
