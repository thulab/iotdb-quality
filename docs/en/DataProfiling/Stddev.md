# Stddev

## Usage

This function is used to calculate the overall standard deviation of a single column of numeric data.

**Name:** STDDEV

**Input Series:** Only supports a single input sequence, the type is INT32 / INT64 / FLOAT / DOUBLE

**Output Series:** Standard deviation
