# Dtw

## 函数简介

本函数用于根据DTW距离计算方法，计算单列数值型数据的对称度。

**函数名：** Symmetric

**输入序列：** 仅支持一个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE。

**输出序列：** 输出单个序列，类型为DOUBLE。序列仅包含一个时间戳为0、值为输入时间序列的对称度。

**提示：** 

+ 如果某行数据中包含空值、缺失值或`NaN`，该行数据将会被忽略；
+ 如果数据中所有的行都被忽略，函数将会输出0。
