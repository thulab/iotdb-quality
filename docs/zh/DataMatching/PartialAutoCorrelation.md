# PACF

## 函数简介

本函数通过求解Yule-Walker方程，计算序列的偏自相关系数。

**函数名：** PACF

**输入序列：** 仅支持单个输入序列，类型为 INT32 / INT64 / FLOAT / DOUBLE。

**参数：**

+ `lag`：最大滞后阶数。默认值为$\min(10\log_{10}n,n-1)$​，$n$​表示数据点个数。​

**输出序列**：输出单个序列，类型为DOUBLE。
